Data Exploration:

Data Preprocessing

For our data exploration, we analyzed a time series dataset of tweets collected between 2016 and 2022 which contains misinformation about De Lima. We used this to see if there is a relationship between the frequency of misinforming tweets and the current events that concerned De Limaâ€™s alleged connection to drug lords. To prepare the gathered dataset for analysis, we have done the following preprocessing steps:

Handling missing values

Since we mainly use date in the data exploration, we will only check the date of the tweet posted and the date of the account created. This is not an issue as Twitter automatically attaches the timestamp with the tweet itself. Twitter also shows publicly when you joined Twitter. However, Twitter does not show the day when you joined Twitter. To handle this, we only took account the month and year of the tweet posted to stay consistent with the other date.

Handling outliers

We removed outliers in the data especially with tweets that managed to sneak in that have dates outside 2016-2022. We used the Find method in Google Sheets if there are tweets outside in the allowed range. We have also computed the standard deviation to check if there are tweets that are 3 standard deviations away from the mean. If there is such data, we will remove the outlying data.

Ensuring Data Consistency

We checked the dates if there are dates that have other date formats (MM/DD/YY instead of DD/MM/YY). If there are we simply switch the format to the intended date format.

Normalization and Stadardization

Data Conversion:

We have extracted the month and year of dates via MID(index, length_of_substring) command in Google Sheets which gets a substring from our string and VALUE(str) which converts a string to a number if possible.We then converted the data to be a float by Year + Month/12 so we can treat it as a number we can graph, and we can make sense of.

a. Month: Month has been extracted via MID(4, 2) which gets the character 4 to 5.

b. Year: Year has been extracted via MID(7, 2) which gets the character 7 to 8.

Note: Timestamp has DD/MM/YY so MM is characters 4 and 5 while YY is characters 7 and 8.


Normalization and Standardization:

We normalized and standardized our dataset so we can compare our data much easier:

a. Normalization: The tweet date has been normalized from 0 to 1, where 0 is the oldest and 1 is the highest. This allows us to easily see if thereâ€™s an outlier since we reduced our range to 0 and 1.

b. Standardization: The tweet date has been standardized to see if there are outliers in the dataset. If a data is 3 standard deviations away from the mean, we will remove the data from our dataset.

Linear Interpolation

Interpolation was used to take account for the missing values in the time series dataset. This allows us to see the time as a continuous value rather than scattered dots along the graph:

a. Importance of Interpolation: Using interpolation is important as we can estimate the missing values in the dataset due to data collection issues or gaps in the time series. By doing this, we can have a much more continuous dataset.

b. Methodology: We used linear interpolation to estimate the missing values by checking the neighboring data points. We used matplotlib.pyplot and numpy to do linear interpolation and display the graph for us.

Binning

We used binning to group the time series data into months from 2016 to 2022:

a. Purpose of Binning: Binning allows us to easily find trends and patterns in the data. Since we grouped the data into months, we can see what months have the highest or lowest frequency from 2016 to 2022.

b. Methodology: We divided the dataset into 12 months per year. We figured that this will allow us to connect the dots between misinformation surge and current events about De Lima without having the bin to be too small and having the divided dataset to be too large to analyze more in-depth.

Visualization

Number of Tweets Containing False Information about De Lima:


We have recorded the number of tweets spreading misinformation against Leila De Lima from our sample. Then, we created a graph to visualize the frequency of misinforming tweets per year, though we further divide the bars by months and indicate what month it is based on the color listed on the legends.

Upon visual inspection, the month with the highest frequency of misinformation is March 2017. Usually, we treat this as an outlier but this has not been changed because of our hypothesis that sees if an event happened or a news reported which led to a misinformation surge in Twitter.

This lines up with our initial hypothesis because Leila De Lima was arrested on February 24, 2017 and De Limaâ€™s case has caught the attention of the European Parliament on March 16, 2017. The second mode, August 2016, also lines up with our initial hypothesis because this was the time where Duterte accused De Lima of having connections with drugs, which happened on August 17, 2016.

Number of Accounts that Tweeted False Information about De Lima:


Upon visual inspection, there is an insignificant difference in amount between the accounts created that spread information between 2009-2015 and 2016-2022. However, the mode of the frequency of accounts created is March 2017 which is also the mode of the frequency of the misinforming tweets.

The Correlation between the Date of Tweets and the Date of Account Created:


Due to modes lining up in the frequency of tweets to the frequency of accounts created, we have plotted the creation date of the misinforming tweet against the creation date of the account spreading misinformation to see if there is a surge of new Twitter accounts to increase the volume of misinformation that have been spread. However, upon visual inspection on the graph, old accounts, or accounts made before 2016, have also spread misinformation.

Regardless of the insight gained from the visual inspection, we computed the Pearson correlation of the two variables which is r = 0.512 (high degree). This means that despite there being several old accounts spreading misinformation, there is still a significant amount of new accounts made to spread misinformation.
